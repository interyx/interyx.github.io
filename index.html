My Mission

My mission is to use my skills and knowledge to make a positive impact on the world. As a developer, I believe that I have a responsibility to create meaningful work that benefits society, whether it's through solving complex problems, improving processes, or designing products and services that improve people's lives.

I am passionate about web development, networking, design, and systems structures, and I strive to incorporate these values into everything I do. I am dedicated to continuous learning and growth, and I believe that collaboration and teamwork are essential for achieving success.

Whether it's working with clients to understand their needs, collaborating with colleagues to develop innovative solutions, or staying up-to-date with the latest trends and technologies in my field, I am always looking for ways to improve and make a positive impact.

I am excited to share my portfolio with you and hope that my work reflects my mission to create positive change through my profession.

SKILLS

Bootstrap is a popular open-source front-end framework that helps developers build responsive, mobile-first web applications quickly and easily.

I used React to help build the frontend for a full-stack fantasy football draft management app which included routes, logins, tight integration with backend API calls and front- and backend team management. I'm also working on a full-stack home inventory app to help find and manage items, collections, chores, recipes, and more inside the home.

I'm experienced using Python to solve small and large challenges. I built a text adventure game engine in Python using a Parsing Expression Grammar to read and interpret user commands, and worked with Django on the backend of a full-stack web app for managing fantasy football drafts.

I love C. I've used it to program an embedded board with a variety of features including temperature sensor with alarm for high/low temperatures and basic REST API as well as low-level image manipulation and multithreading.
